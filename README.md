# LowQuality

A minimalist landing page for my clothing brand, **LowQuality**.

Built around a monochrome design language with a focus on simplicity, contrast, and clean product presentation. The site showcases the brand philosophy, featured pieces, and the visual identity of the first collection.

🚧 **Status:** In active development  
🗓️ **Planned Launch:** December 2026

### Live Demo

https://lowquality-store.netlify.app/

## Features

- Minimal black-and-white aesthetic
- Responsive landing page
- Featured product showcase
- Brand story and philosophy
- Contact section
- Collection preview

## Tech Stack

- HTML5
- CSS3
